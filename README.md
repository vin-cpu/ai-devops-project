# DevOps for AI - Mini Project

## Overview
This project demonstrates a simple DevOps pipeline for deploying a machine learning model.

## Tools Used
- Scikit-learn
- FastAPI
- Docker
- GitHub Actions

## Workflow
1. Train ML model
2. Save model
3. Create API
4. Dockerize application
5. Automate using CI/CD

## Outcome
A production-ready AI model deployment pipeline.
